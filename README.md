# book-blog

A blog where I can write about the books I'm reading.

## Usage

1. Clone this repo.
2. `cd book-blog`
3. `yarn install`
4. `npm start`
5. The app will now be running on localhost:3000

## Technologies Used.

-   Yarn (package management)
-   NodeJs
    -   Express
    -   Cors
    -   Nodemon
    -   Bcrypt
    -   Dotenv
    -   pg-promise
